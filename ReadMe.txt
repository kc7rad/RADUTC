RADUTC Readme

RADUTC is a simple Windows program to display the current UTC on a semi-transparent frameless, borderless form.

the form may be moved on the desktop by left-clicking and dragging the RADUTC form anywhere on the desktop.

All program options are available by right-clicking on the RADUTC form.  
Available options include font color, background color, focused opacity and unfocused opacity.

By Ken Linder (KC7RAD)
